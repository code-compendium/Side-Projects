# Tic-Tac-Toe game

Simple, basic tic tac toe game with local storage to save the score.

## Features:

- Keep track of score, save in localstorage
- Reset score
- Reset board
- Keep track of whose turn it is
- Keep track of the amount of wins/losses/draws
- Keep track of the amount of games played
- Randomize the starting player
- Let players pick their own names

## Tech stack:

React + Vite, CSS
